‚ú® Task Objectives

This project builds a multi-model classification system to predict movie genres based on plot summaries. It integrates both classical machine learning and modern deep learning techniques to evaluate model performance and classification accuracy.

  • Develop a system to classify movie plots into genres using multiple models.
  • Compare performance of classical and deep learning approaches.
  • Handle class imbalance and support custom user input for predictions.

üß† Models Implemented

  • Random Forest with TF-IDF vectorization and handcrafted features
  • BERT + Logistic Regression (shallow neural model on BERT embeddings)
  • BERT + MLP (deep neural network using Keras)

üõ† Steps to Run the Project

  1. Clone the repository: 
    git clone https://github.com/your-username/movie-genre-classification.git
cd movie-genre-classification
  2. Download and prepare the dataset from Kaggle. Place the following in Genre Classification Dataset/:
    • train_data.txt
    • test_data.txt
    • test_data_solution.txt
  3. Download trained models and place in models/:
    • bert_logreg_model.pkl
    • bert_mlp_model.h5
    • bert_mlp_encoder.pkl
    • random_forest_model.pkl
  4. Install requirements: 
    pip install -r requirements.txt
  5. Run the system: 
    python model.py

üìä Outputs

Models generate confusion matrices, classification reports, and misclassified sample logs. Results and saved models are available in the models/ directory. You can also provide your own movie plot to get genre predictions.
