🌿 Project Overview

LeafScan is a real-time AI-based application that detects plant diseases from leaf images using deep learning. Built with MobileNetV2 and deployed via Streamlit, the model enables fast, accurate diagnosis to support smart agriculture and research in plant pathology.

🎯 Problem Statement

Plant diseases cause 20–40% of global crop losses annually, resulting in over $220 billion in economic damage. Timely diagnosis is often unavailable due to limited access to expert pathologists, especially in rural farming regions.

💡 Our Solution

  • 91% accurate classification of 38+ plant diseases
  • Deep learning with MobileNetV2 and focal loss for class imbalance
  • User-friendly Streamlit interface accessible on any device
  • Real-time prediction (<2 seconds per image)
  • Optimized model (<50MB) for mobile and edge deployment

🧠 Model Architecture

  • Input Layer: 224x224x3 RGB image
  • MobileNetV2 (pretrained on ImageNet)
  • Global Average Pooling
  • Dense Layer (128 units, ReLU)
  • Dropout (0.5)
  • Output Layer: 38 classes, Softmax

⚙️ Features

  • Streamlit web app with drag-and-drop interface
  • Live confidence scores for predictions
  • Error handling for invalid images
  • Analytics dashboard with precision, recall, F1-score
  • Confusion matrix and class distribution visualizations

📂 Dataset

The dataset includes 38 plant disease classes and healthy samples, across crops such as apple, corn, grape, tomato, and strawberry. It is split 80/20 into training and validation sets.

🚀 Quick Start

# Run the web app
streamlit run app/streamlit_app.py

# Predict from CLI
python src/predict.py path/to/image.jpg

📈 Model Performance

  • Accuracy: 91%
  • Macro F1 Score: 89%
  • Precision: 92%
  • Recall: 91%
  • Top Classes: Corn Rust (F1 0.98), Apple Black Rot, Strawberry Scorch
